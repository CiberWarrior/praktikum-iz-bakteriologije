┌────────────────────────────────────────────────────────────────────────────┐
│ KORAK 1: Otvorite FileZilla Client                                        │
└────────────────────────────────────────────────────────────────────────────┘

  FileZilla izgleda ovako:
  
  ┌──────────────────────────────────────────────────────────────────────┐
  │ File  Edit  View  Transfer  Server  Bookmarks  Help                 │
  ├──────────────────────────────────────────────────────────────────────┤
  │ Host: [____________]  User: [______]  Pass: [______]  Port: [___]   │
  │                                                    [Quickconnect]    │
  ├──────────────────────────────────────────────────────────────────────┤
  │ Message log (status poruke)                                          │
  ├──────────────────────────────────────────────────────────────────────┤
  │  LOCAL SITE (vaše računalo)  │  REMOTE SITE (server)                │
  │  /Users/renchi/Desktop/...   │  /public_html/                       │
  │  [direktoriji i datoteke]    │  [server direktoriji]                │
  ├──────────────────────────────────────────────────────────────────────┤
  │ Transfer queue (red čekanja za upload)                               │
  └──────────────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────────────────────┐
│ KORAK 2: Povežite se na FTP server                                        │
└────────────────────────────────────────────────────────────────────────────┘

  U gornja polja unesite:
  
  ┌──────────────────────────────────────────────────────────────────┐
  │ Host:     ftp.biol.pmf.hr                                        │
  │ Username: [ZATRAŽITE OD ADMIN]                                   │
  │ Password: [ZATRAŽITE OD ADMIN]                                   │
  │ Port:     21 (FTP) ili 22 (SFTP)                                 │
  └──────────────────────────────────────────────────────────────────┘
  
  ➜ Kliknite [Quickconnect]

┌────────────────────────────────────────────────────────────────────────────┐
│ KORAK 3: Navigirajte do pravih lokacija                                   │
└────────────────────────────────────────────────────────────────────────────┘

  LIJEVA STRANA (Local site):
  ➜ Navigirajte do:
    /Users/renchi/Desktop/praktikum-iz-bakteriologije/dist/
    
  DESNA STRANA (Remote site):
  ➜ Navigirajte do web root direktorija:
    Najčešće: /public_html/ ili /www/ ili /htdocs/
    
  ⚠️  VAŽNO: Pitajte administratora gdje točno!

┌────────────────────────────────────────────────────────────────────────────┐
│ KORAK 4: Upload svih datoteka                                             │
└────────────────────────────────────────────────────────────────────────────┘

  1. Na LIJEVOJ strani:
     - Otvorite dist/ folder
     - Pritisnite Ctrl+A (Mac: Cmd+A) da označite SVE datoteke
     
  2. Upload:
     METODA A: Desni klik → Upload
     METODA B: Povucite datoteke s lijeve na desnu stranu (drag & drop)
     
  3. Pratite progress:
     - U donjem dijelu vidite queue
     - Successful transfers = ✓
     - Failed transfers = ⚠️ (pokušajte ponovno)

┌────────────────────────────────────────────────────────────────────────────┐
│ KORAK 5: Provjerite upload                                                │

     • Broj slika: 105 slika

╔════════════════════════════════════════════════════════════════════════════╗
║                      RJEŠAVANJE PROBLEMA                                   ║
╚════════════════════════════════════════════════════════════════════════════╝

  ❌ PROBLEM: Ne mogu se povezati
     ✓ Provjerite FTP kredencijale
     ✓ Pokušajte port 22 (SFTP) umjesto 21 (FTP)
     ✓ Kontaktirajte administratora
     
  ❌ PROBLEM: 404 Not Found
     ✓ Provjerite da su datoteke u root direktoriju
     ✓ Provjerite putanje (case-sensitive!)
     
  ❌ PROBLEM: Slike se ne vide
     ✓ Uploadajte ponovno Images/ direktorij
     ✓ Provjerite da je naziv direktorija "Images" (veliko I!)
     
  ❌ PROBLEM: CSS ne radi
     ✓ Uploadajte ponovno _astro/ direktorij
     ✓ Obrisite browser cache
     
  ❌ PROBLEM: Upload se prekida
     ✓ Uploadajte direktorij po direktorij
     ✓ Prvo manje datoteke, pa veće
