## Bojenje acidorezistentnih bakterija po Ziehl-Neelsenu

<div style="background: linear-gradient(135deg, #e8f4fd 0%, #d1ecf1 100%); border-radius: 15px; padding: 25px; margin: 20px 0;">
  
  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #2196f3;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🦠</span>
      <h3 style="margin: 0; color: #1565c0; font-size: 1.3rem; font-weight: 600;">Acidorezistentne bakterije</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Neke bakterije posebno vrste iz roda <em>Mycobacterium</em> (<em>M. tuberculosis</em>, <em>M. lepreae</em>, <em>M. smegmatis</em>) otporne su na bojenja bazičnim bojilima jer oko stanice posjeduju omotač od fosfolipida i voska, koji ne propušta boje u stanice.
    </p>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #ff9800;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🔥</span>
      <h3 style="margin: 0; color: #f57c00; font-size: 1.3rem; font-weight: 600;">Princip metode</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Ta se ovojnica mora učiniti propusnom, i to grijanjem preparata prelivenog bojom (karbol fuksin) nad otvorenim plamenom.
    </p>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #4caf50;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">📋</span>
      <h3 style="margin: 0; color: #388e3c; font-size: 1.3rem; font-weight: 600;">Postupak bojenja</h3>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
      <h4 style="margin: 0 0 10px 0; color: #388e3c; font-size: 1.1rem;">1. Grijanje preparata</h4>
      <p style="margin: 0; color: #2d3748; line-height: 1.6;">
        Preparat se drži nad plamenom, a kada se iz boje počne dizati para, odmakne se od plamena te ponovno primakne kada prestane, a cijeli postupak traje 5 minuta. Treba paziti da boja ne provre, da ne ispari, a preparat se posuši, tako da se po potrebi boja dolijeva.
      </p>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
      <h4 style="margin: 0 0 10px 0; color: #388e3c; font-size: 1.1rem;">2. Ispranje kiselim alkoholom</h4>
      <p style="margin: 0; color: #2d3748; line-height: 1.6;">
        Tako tretiran sada obojen preparat ispire se kiselim alkoholom, sve dok ne prestane izlaziti crvena boja, te ispere vodom iz vodovoda.
      </p>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px;">
      <h4 style="margin: 0 0 10px 0; color: #388e3c; font-size: 1.1rem;">3. Kontrabojenje</h4>
      <p style="margin: 0; color: #2d3748; line-height: 1.6;">
        Na tako ispran preparat doda se metilensko modrilo kao kontrastna boja i ostavi 2 min. Nakon ispiranja vodom iz vodovoda preparat se posuši filter-papirom i mikroskopira.
      </p>
    </div>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #9c27b0;">

    <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
      <li><strong>Acidorezistentne bakterije</strong>: Ostaju crveno obojene</li>
      <li><strong>Ostale bakterije</strong>: Obojene su plavo</li>
    </ul>
  </div>

### Ključne karakteristike Ziehl-Neelsen metode

- **Specifičnost**: Namijenjena acidorezistentnim bakterijama
- **Toplinska obrada**: Ključna za propusnost voskane ovojnice
- **Kiselo ispranje**: Razlikuje acidorezistentne od ostalih bakterija
- **Klinički značaj**: Osnova za dijagnozu tuberkuloze
