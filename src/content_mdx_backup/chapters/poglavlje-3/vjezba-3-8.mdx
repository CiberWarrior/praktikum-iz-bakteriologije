## Vježba 3.8 - Negativno bojenje kapsula

<div style="background: linear-gradient(135deg, #e8f4fd 0%, #d1ecf1 100%); border-radius: 15px; padding: 25px; margin: 20px 0;">
  
  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #2196f3;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🎯</span>
      <h3 style="margin: 0; color: #1565c0; font-size: 1.3rem; font-weight: 600;">Cilj vježbe</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Savladavanje tehnike negativnog bojenja za dokazivanje bakterijskih kapsula.
    </p>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #ff9800;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🧪</span>
      <h3 style="margin: 0; color: #f57c00; font-size: 1.3rem; font-weight: 600;">Materijali i reagensi</h3>
    </div>
    <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
      <li>Čiste predmetnice (2 komada)</li>
      <li>5%-tna otopina nigrozina (ili crni tuš)</li>
      <li>Čista kultura bakterija s kapsulama</li>
      <li>Bakteriološka ušica</li>
      <li>Karbol fuksin ili metilensko modrilo (za naknadno bojenje)</li>
      <li>Mikroskop s imerzionim objektivom</li>
    </ul>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #4caf50;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">📋</span>
      <h3 style="margin: 0; color: #388e3c; font-size: 1.3rem; font-weight: 600;">Postupak</h3>
    </div>
    <ol style="margin: 0; padding-left: 20px; color: #2d3748;">
      <li style="margin-bottom: 8px;"><strong>Priprema razmaza</strong> - Na jedan kraj suhe, čiste predmetnice kapnuti kap <strong>5%-tne otopine nigrozina</strong> (ili crnog tuša)</li>
      <li style="margin-bottom: 8px;"><strong>Suspendiranje bakterija</strong> - U kapljicu suspendirati bakteriološkom ušicom materijal čiste kulture bakterija</li>
      <li style="margin-bottom: 8px;"><strong>Razmaz</strong> - Drugom predmetnicom dotaknuti jednim krajem kapljicu nigrozina</li>
      <li style="margin-bottom: 8px;"><strong>Čekanje širenja</strong> - Čekati da se kapljica proširi uzduž svoga ruba</li>
      <li style="margin-bottom: 8px;"><strong>Razmazivanje</strong> - Razvući bojilo s bakterijskom kulturom preko površine prve predmetnice u što tanjem razmazu</li>
      <li style="margin-bottom: 8px;"><strong>Sušenje</strong> - Osušiti na zraku, <strong>ne fiksirati plamenom</strong></li>
      <li style="margin-bottom: 8px;"><strong>Prvo promatranje</strong> - Pripremljen preparat promatrati imerzijskim objektivom</li>
      <li style="margin-bottom: 0;"><strong>Naknadno bojenje</strong> - Da bi se kapsula razlikovala od bakterija, osušeni preparat naknadno obojiti jednom bojom (najčešće <strong>karbol fuksinom</strong> ili <strong>metilenskim modrilom</strong>)</li>
    </ol>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #9c27b0;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🔬</span>
      <h3 style="margin: 0; color: #7b1fa2; font-size: 1.3rem; font-weight: 600;">Rezultati</h3>
    </div>
    <p style="margin: 0 0 15px 0; color: #2d3748; line-height: 1.6;">
      U naknadno obojenu tuš-preparatu vidjet će se:
    </p>
    <ul style="margin: 0 0 15px 0; padding-left: 20px; color: #2d3748;">
      <li><strong>Kapsule kao bezbojni omotač</strong> (prozirne)</li>
      <li><strong>Protoplazma bakterija</strong> bit će obojena ovisno o boji koja je upotrijebljena</li>
      <li><strong>Tamna pozadina</strong> zbog nigrozina</li>
    </ul>

    </div>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; border-left: 4px solid #607d8b;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">✅</span>
      <h3 style="margin: 0; color: #455a64; font-size: 1.3rem; font-weight: 600;">Zaključak</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Negativno bojenje kapsula omogućuje vizualizaciju bakterijskih kapsula bez oštećenja njihove strukture, jer se bojilo ne veže direktno na kapsule već stvara tamnu pozadinu koja ih čini vidljivima.
    </p>
