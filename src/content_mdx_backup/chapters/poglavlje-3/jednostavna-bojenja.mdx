## Jednostavna bojenja

<div style="background: linear-gradient(135deg, #e8f4fd 0%, #d1ecf1 100%); border-radius: 15px; padding: 25px; margin: 20px 0;">
  
  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #2196f3;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🔬</span>
      <h3 style="margin: 0; color: #1565c0; font-size: 1.3rem; font-weight: 600;">Načelo jednostavnih bojenja</h3>
    </div>
    <p style="margin: 0 0 15px 0; color: #2d3748; line-height: 1.6;">
      Jednostavna bojenja su osnovne tehnike bojenja bakterija koje koriste jednu boju za vizualizaciju bakterijskih stanica pod mikroskopom.
    </p>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Jednostavna bojenja koriste jednu boju koja se vezuje na bakterijsku stanicu i omogućuje vizualizaciju:
    </p>
    <ul style="margin: 10px 0 0 0; padding-left: 20px; color: #2d3748;">
      <li>Oblika bakterija (kuglaste, štapićaste, spiralne)</li>
      <li>Rasporeda bakterija (pojedinačne, u lancima, u grozdovima)</li>
      <li>Prisutnosti određenih struktura (kapsule, flagela)</li>
    </ul>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #ff9800;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🎨</span>
      <h3 style="margin: 0; color: #f57c00; font-size: 1.3rem; font-weight: 600;">Najčešće korištene boje</h3>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
      <h4 style="margin: 0 0 10px 0; color: #f57c00; font-size: 1.1rem;">1. Karbolfuksin</h4>
      <p style="margin: 0 0 8px 0; color: #2d3748;"><strong>Sastav:</strong> Fuchsin u fenolnoj otopini</p>
      <p style="margin: 0 0 8px 0; color: #2d3748;"><strong>Boja:</strong> Crvena</p>
      <p style="margin: 0; color: #2d3748;"><strong>Primjena:</strong> Općenito bojenje, acidorezistentne bakterije</p>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
      <h4 style="margin: 0 0 10px 0; color: #f57c00; font-size: 1.1rem;">2. Gencjan violet</h4>
      <p style="margin: 0 0 8px 0; color: #2d3748;"><strong>Sastav:</strong> Kristalni violet</p>
      <p style="margin: 0 0 8px 0; color: #2d3748;"><strong>Boja:</strong> Ljubičasta</p>
      <p style="margin: 0; color: #2d3748;"><strong>Primjena:</strong> Gram pozitivne bakterije, općenito bojenje</p>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px;">
      <h4 style="margin: 0 0 10px 0; color: #f57c00; font-size: 1.1rem;">3. Metilensko plavo</h4>
      <p style="margin: 0 0 8px 0; color: #2d3748;"><strong>Sastav:</strong> Metilenski plavi</p>
      <p style="margin: 0 0 8px 0; color: #2d3748;"><strong>Boja:</strong> Plava</p>
      <p style="margin: 0; color: #2d3748;"><strong>Primjena:</strong> Općenito bojenje, metahromatske granule</p>
    </div>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #4caf50;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">📋</span>
      <h3 style="margin: 0; color: #388e3c; font-size: 1.3rem; font-weight: 600;">Postupak jednostavnog bojenja</h3>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 10px;">
      <h4 style="margin: 0 0 8px 0; color: #388e3c; font-size: 1.1rem;">1. Priprema preparata</h4>
      <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
        <li>Fiksacija bakterija na stakalcu</li>
        <li>Sušenje na zraku</li>
      </ul>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 10px;">
      <h4 style="margin: 0 0 8px 0; color: #388e3c; font-size: 1.1rem;">2. Bojenje</h4>
      <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
        <li>Nanositi boju 1-2 minute</li>
        <li>Isprati destiliranom vodom</li>
      </ul>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px; margin-bottom: 10px;">
      <h4 style="margin: 0 0 8px 0; color: #388e3c; font-size: 1.1rem;">3. Sušenje</h4>
      <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
        <li>Sušiti na zraku ili blagim zagrijavanjem</li>
      </ul>
    </div>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px;">
      <h4 style="margin: 0 0 8px 0; color: #388e3c; font-size: 1.1rem;">4. Mikroskopiranje</h4>
      <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
        <li>Pod imerzionim objektivom</li>
      </ul>
    </div>
  </div>

      <li>Niska cijena</li>
    </ul>
