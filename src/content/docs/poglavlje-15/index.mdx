# Poglavlje 15: Bakteriološko ispitivanje kvalitete namirnica

## Predgovor

U bakteriologiji živežnih namirnica upotrebljavaju se brzi testovi kojima se može ustanoviti higijenska vrijednost namirnice, pa i ukupan broj bakterija u njima.
