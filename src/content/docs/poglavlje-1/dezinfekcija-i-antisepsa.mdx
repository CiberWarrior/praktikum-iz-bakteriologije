    Tijekom i nakon svakog praktikuma potrebno je obaviti dezinfekciju radnih površina i antisepsu ruku.
  </p>

<div style="background: linear-gradient(135deg, #e8f5e8 0%, #f1f8e9 100%); border-radius: 15px; padding: 20px; margin: 20px 0; border: 2px solid #27ae60;">
  <div style="display: flex; align-items: center; margin-bottom: 15px;">
    <span style="font-size: 24px; margin-right: 12px;">🧽</span>
    <h4 style="margin: 0; color: #1e7e34; font-weight: 600; font-size: 18px;">Dezinfekcija</h4>
  </div>
  <div style="background: rgba(255,255,255,0.9); padding: 15px; border-radius: 10px;">
    <p style="margin: 0 0 10px 0; color: #2d3748; line-height: 1.6;">
      <strong>Dezinfekcija</strong> <em style="color: #6c757d;">(lat. inficiens – koji truje)</em> je tretman kemijskim agensima – dezinficijensima radi uništavanja bakterija i drugih mikroba u zraku, na radnim površinama i predmetima.
    </p>
    <div style="background: #f8f9fa; padding: 12px; border-radius: 8px; border-left: 3px solid #27ae60;">
      <strong style="color: #1e7e34; font-size: 14px;">Dezinficijensi:</strong>
      <div style="margin-top: 5px; color: #495057; font-size: 14px; line-height: 1.5;">
        Halogeni, derivati joda i klora, metanol, etanol, izopropanol, fenol i derivati fenola, vodikov peroksid
      </div>
    </div>
  </div>
</div>

<div style="background: linear-gradient(135deg, #fff3cd 0%, #ffeaa7 100%); border-radius: 10px; padding: 20px; margin: 20px 0; border: 1px solid #e17055;">
  <div style="display: flex; align-items: center; margin-bottom: 15px;">
    <span style="font-size: 20px; margin-right: 10px;">⚠️</span>
    <strong style="color: #d63031; font-size: 16px;">Postupak kod izlijevanja</strong>
  </div>
  <p style="margin: 0; color: #2d3436; line-height: 1.6;">
    Prolivene suspenzije bakterijskih kultura odmah treba prekriti papirnatim ručnikom i potom ga natopiti dezinficijensom, ostaviti tako <strong>petnaest minuta</strong> i onda odložiti na predviđeno odlagalište kontaminiranog materijala.
  </p>
</div>

<div style="background: linear-gradient(135deg, #e1f5fe 0%, #b3e5fc 100%); border-radius: 15px; padding: 20px; margin: 20px 0; border: 2px solid #0288d1;">
  <div style="display: flex; align-items: center; margin-bottom: 15px;">
    <span style="font-size: 24px; margin-right: 12px;">🖐️</span>
    <h4 style="margin: 0; color: #01579b; font-weight: 600; font-size: 18px;">Antisepsa</h4>
  </div>
  <div style="background: rgba(255,255,255,0.9); padding: 15px; border-radius: 10px;">
    <p style="margin: 0 0 10px 0; color: #2d3748; line-height: 1.6;">
      <strong>Antisepsa</strong> podrazumijeva dezinfekciju živog tkiva, npr. površina tijela, ruke, usna šupljina, antisepticima.
    </p>
    <div style="background: #f8f9fa; padding: 12px; border-radius: 8px; border-left: 3px solid #0288d1;">
      <strong style="color: #01579b; font-size: 14px;">Antiseptici:</strong>
      <div style="margin-top: 5px; color: #495057; font-size: 14px; line-height: 1.5;">
        Kalijev permanganat, vodikov peroksid, komercijalni antiseptici
      </div>
    </div>
  </div>
</div>

<div style="background: linear-gradient(135deg, #fff5f5 0%, #fed7d7 100%); border-radius: 10px; padding: 18px; margin: 20px 0; border-left: 4px solid #e53e3e;">
  <div style="display: flex; align-items: center; margin-bottom: 8px;">
    <span style="font-size: 18px; margin-right: 8px;">⚡</span>
    <strong style="color: #c53030;">Važno ograničenje</strong>
  </div>
  <p style="margin: 0; color: #2d3748; line-height: 1.5; font-weight: 500;">
    Postupcima dezinfekcije i antisepse <strong>ne postiže se potpuna sterilizacija</strong>, jer se ne uništavaju bakterijske endospore.
  </p>
</div>
