  <section class="cards-grid">
    <article class="card-sheen">
      <h3 class="card-sheen__title">Cilj vježbe</h3>
      <p>Dokazivanje sveprisutnosti mikroorganizama u okolišu kroz uzorkovanje iz različitih izvora i promatranje rasta kolonija.</p>
    </article>
    
    <article class="card-sheen">
      <h3 class="card-sheen__title">Potrebni materijali</h3>
      <ul class="list-ticks">
        <li>Sterilni štapići za uzimanje brisa</li>
        <li>Petrijeve zdjelice s hranjivim agarom</li>
        <li>Sterilna destilirana voda s 0,3% NaCl</li>
        <li>Inkubator (22 °C)</li>
      </ul>
    </article>
  </section>

  <section class="timeline">
    <h2>Koraci</h2>
    
    <details class="t-step" open>
      <summary>
        <span class="t-step__num">1</span>
        Priprema uzorkovanja
      </summary>
      <div class="t-step__body">
        <p>Sterilni štapić za uzimanje brisa umočiti u sterilnu destiliranu vodu s dodatkom 0,3% NaCl. Uzeti brisove đona cipela, radnih površina, kvaka, stuba i sl. Premazati štapićem površinu Petrijeve zdjelice s hranjivim agarom i inkubirati na 22 °C / 72h.</p>
      </div>
    </details>

    <details class="t-step" open>
      <summary>
        <span class="t-step__num">2</span>
        Uzorkovanje zraka
      </summary>
      <div class="t-step__body">
        <p>Radi dokazivanja mikroorganizama koji se nalaze oko nas u zraku, Petrijevu zdjelicu s hranjivim agarom otvoriti i ostaviti bez poklopca izloženu zraku 20 min. Petrijeve zdjelice izložiti u predavaonici, praktikumu, hodniku, na prozoru.</p>
      </div>
    </details>
  </section>

  <section class="cards-grid">
    <article class="card-sheen">
      <h3 class="card-sheen__title">Rezultati inkubacije</h3>
      <p>Nakon inkubacije (22 °C / 72h) na pločama će porasti kolonije bakterija (slika 10), kvasaca i plijesni, a njihov broj zavisi od mjesta uzorkovanja. Na mjestima sakupljanja i zadržavanja većeg broja ljudi, u zatvorenim prostorijama, na cesti u centru grada bit će znatno veći nego u šumi, parkovima i sl.</p>
    </article>
  </section>

  <section class="cards-grid">
    <article class="card-sheen">
      <h3 class="card-sheen__title">Karakteristike kolonija</h3>
      <p>Obratiti pozornost na oblik i izgled ruba bakterijskih kolonija:</p>
      
      <div style="margin-top: 1rem;">
        <h4 style="margin: 0 0 0.5rem 0; color: var(--brand-700);">🔵 Oblik kolonija</h4>
        <p style="margin: 0 0 1rem 0; font-size: 0.9rem;">Gledano odozgo oblik kolonija pretežno je okrugao, a rubovi glatki, valoviti, izrezuckani ili čupavi.</p>
        
        <h4 style="margin: 0 0 0.5rem 0; color: var(--brand-700);">📐 Profil kolonija</h4>
        <p style="margin: 0 0 1rem 0; font-size: 0.9rem;">Gledane sa strane, kolonije mogu biti konveksne s udubljenjem ili ispupčenjem u sredini, ili izgleda okrugle ploče.</p>
        
        <h4 style="margin: 0 0 0.5rem 0; color: var(--brand-700);">✨ Površina kolonija</h4>
        <p style="margin: 0; font-size: 0.9rem;">Površina kolonija može biti glatka ili hrapava, katkada s raznim specifičnim crtežima, a kolonije mogu biti različito obojene ili prozirne.</p>
      </div>
    </article>
  </section>
