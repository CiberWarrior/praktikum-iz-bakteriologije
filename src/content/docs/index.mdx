<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Kako koristiti</h3>

1. Kreni od <a href="/predgovor/">Predgovora</a> (kontekst, struktura i sigurnost).
2. Otvori <a href="/predgovor/#pregled-poglavlja">Pregled poglavlja</a> i izaberi temu.
3. Slijedi protokol → odradi vježbu → usporedi rezultat sa slikama.
4. Bilježi opažanja i ponovi ključne korake.

:::tip[Preporuka]
Najbolje rezultate daje kombinacija čitanja i izvođenja vježbi u laboratoriju.
:::
