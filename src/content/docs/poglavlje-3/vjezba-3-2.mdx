---
title: Vježba 3.2 - Bojenje bakterija gencijanom violet
description: Osnovno bojenje bakterija pomoću gencijana violet
---

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Cilj vježbe</h3>

Savladavanje tehnike osnovnog bojenja bakterija pomoću gencijana violet.

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Materijali i reagensi</h3>

- Pripremljeni bakterijski preparat
- Gencijana violet
- Voda iz vodovoda
- Imerziono ulje
- Mikroskop s imerzionim objektivom

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Postupak</h3>

1. Preliti **gencijanom violet** bakteriološki preparat
2. Bojiti **2-3 minute**
3. Isprati vodom iz vodovoda
4. Osušiti na zraku
5. Kapnuti na razmaz kap imerzionog ulja
6. Promatrati imerzionim objektivom

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Rezultati</h3>

Bakterije su obojene u tamno plavu/ljubičastu boju zbog gencijana violet.

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Zaključak</h3>

Gencijana violet je osnovni bazični bojilo koji omogućuje jasnu vizualizaciju bakterijskih stanica.
