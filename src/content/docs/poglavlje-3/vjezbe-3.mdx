U ovom dijelu praktikuma izvodit ćete 8 različitih vježbi bojenja bakterija koje pokrivaju osnovne i specifične tehnike bojenja.

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Pregled vježbi</h3>

1. **[Bojenje bakterija karbolfuksinom](/poglavlje-3/vjezba-3-1/)**  
   Osnovno bojenje bakterija pomoću karbolfuksina

2. **[Bojenje bakterija gencijanom violet](/poglavlje-3/vjezba-3-2/)**  
   Osnovno bojenje bakterija pomoću gencijana violet

3. **[Bojenje bakterija metilenskim plavim](/poglavlje-3/vjezba-3-3/)**  
   Osnovno bojenje bakterija pomoću metilenskog plavog

4. **[Bojenje bakterija po Gramu](/poglavlje-3/vjezba-3-4/)**  
   Složeno diferencijalno bojenje za razlikovanje Gram-pozitivnih i Gram-negativnih bakterija

5. **[Test s KOH-om](/poglavlje-3/vjezba-3-5/)**  
   Brza metoda za razlikovanje Gram-pozitivnih i Gram-negativnih bakterija

6. **[Bojenje acidorezistentnih bakterija po Ziehl-Neelsenu](/poglavlje-3/vjezba-3-6/)**  
   Specifično bojenje za bakterije roda Mycobacterium

7. **[Bojenje bakterijskih endospora](/poglavlje-3/vjezba-3-7/)**  
   Bojenje bakterijskih endospora po Schäffer-Fultonu

8. **[Negativno bojenje kapsula](/poglavlje-3/vjezba-3-8/)**  
   Negativno bojenje za dokazivanje bakterijskih kapsula

<h3 style="color: #096279; font-size: 1.4rem; font-weight: 600; margin: 1.5rem 0 1rem 0;">Napomena</h3>

Prije izvođenja vježbi obavezno pročitajte teorijski dio o metodama bojenja bakterija.
