# Kemijska sterilizacija

Od kemijskih sredstava za sterilizaciju upotrebljavaju se kemikalije kao što su: etilen oksid za sterilizaciju plastike, formaldehid, glutaraldehid, beta-propiolakton. Tim se kemikalijama uništavaju mikroorganizmi te materijal postaje sterilan.
