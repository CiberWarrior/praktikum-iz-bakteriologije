# Bakterije indikatori sanitarnog stanja sredine

Ovo poglavlje obuhvaća bakterije koje se koriste kao indikatori sanitarnog stanja oko inje, posebno fekalnog zagađenja vode i namirnica.

## Sadržaj poglavlja

### [Koliformne bakterije](/poglavlje-8/koliformne-bakterije)
Koliformne bakterije kao glavni indikatori fekalnog zagađenja i prisutnosti patogenih mikroorganizama.

### [Fekalni streptokoki](/poglavlje-8/fekalni-streptokoki)
Fekalni streptokoki (enterokoki) kao dodatni indikatori fekalnih zagađenja sredine.

### [Vježbe 8](/poglavlje-8/vjezbe-8)
Praktične vježbe vezane uz dokazivanje bakterija indikatora sanitarnog stanja.
