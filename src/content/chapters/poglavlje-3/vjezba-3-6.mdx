## Vježba 3.6 - Bojenje acidorezistentnih bakterija po Ziehl-Neelsenu

<div style="background: linear-gradient(135deg, #e8f4fd 0%, #d1ecf1 100%); border-radius: 15px; padding: 25px; margin: 20px 0;">
  
  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #2196f3;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🎯</span>
      <h3 style="margin: 0; color: #1565c0; font-size: 1.3rem; font-weight: 600;">Cilj vježbe</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Savladavanje tehnike bojenja acidorezistentnih bakterija po Ziehl-Neelsenu.
    </p>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #ff9800;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🧪</span>
      <h3 style="margin: 0; color: #f57c00; font-size: 1.3rem; font-weight: 600;">Materijali i reagensi</h3>
    </div>
    <ul style="margin: 0; padding-left: 20px; color: #2d3748;">
      <li>Kultura <em>Mycobacterium phei</em> (nepatogena mikobakterija)</li>
      <li>Lowenstein-Jensenova hranjiva podloga</li>
      <li>Karbol fuksin</li>
      <li>Kiseli alkohol</li>
      <li>Metilensko modrilo</li>
      <li>Voda iz vodovoda</li>
      <li>Pinceta</li>
      <li>Bunsenov plameník</li>
      <li>Mikroskop s imerzionim objektivom</li>
    </ul>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #4caf50;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🦠</span>
      <h3 style="margin: 0; color: #388e3c; font-size: 1.3rem; font-weight: 600;">Priprema kulture</h3>
    </div>
    <p style="margin: 0 0 15px 0; color: #2d3748; line-height: 1.6;">
      Za bojenje u praktikumskoj nastavi preporučuje se primjena nepatogenih mikobakterija, kao što je <em>Mycobacterium phei</em>. <em>Mycobacterium phei</em> uzgaja se na selektivnoj <strong>Lowenstein-Jensenovoj hranjivoj podlozi</strong> na 37°C/72 h.
    </p>
    
    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px;">
      <h4 style="margin: 0 0 10px 0; color: #388e3c; font-size: 1.1rem;">Priprema Lowenstein-Jensenova medija</h4>
      <div style="display: grid; grid-template-columns: 1fr auto; gap: 8px; font-size: 14px;">
        <span style="color: #2d3748;">Sterilizacija medija</span>
        <span style="background: #e74c3c; color: white; padding: 2px 8px; border-radius: 12px; font-size: 12px;">30 min na 121°C</span>
        <span style="color: #2d3748;">Hlađenje</span>
        <span style="background: #3498db; color: white; padding: 2px 8px; border-radius: 12px; font-size: 12px;">Do 30°C</span>
        <span style="color: #2d3748;">Svježa jaja</span>
        <span style="background: #f39c12; color: white; padding: 2px 8px; border-radius: 12px; font-size: 12px;">1L mješavine (20-25 jaja)</span>
        <span style="color: #2d3748;">Starost jaja</span>
        <span style="background: #27ae60; color: white; padding: 2px 8px; border-radius: 12px; font-size: 12px;">Ne starija od 1 tjedan</span>
        <span style="color: #2d3748;">Miješanje</span>
        <span style="background: #9b59b6; color: white; padding: 2px 8px; border-radius: 12px; font-size: 12px;">U steriliziranoj posudi</span>
      </div>
    </div>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #9c27b0;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">📋</span>
      <h3 style="margin: 0; color: #7b1fa2; font-size: 1.3rem; font-weight: 600;">Postupak bojenja</h3>
    </div>
    <ol style="margin: 0; padding-left: 20px; color: #2d3748;">
      <li style="margin-bottom: 8px;"><strong>Prvo bojenje</strong> - <strong>karbol fuksin</strong> nanese se preko razmaza fiksirane bakterijske kulture</li>
      <li style="margin-bottom: 8px;"><strong>Zagrijavanje</strong> - Pridržavajući predmetno stakalce pincetom, preparat se zagrijava s donje strane iznad plamena dok boja ne počne isparavati (oko <strong>15 sekundi</strong>)</li>
      <li style="margin-bottom: 8px;"><strong>Hlađenje</strong> - Preparat se odmakne u stranu da se ohladi</li>
      <li style="margin-bottom: 8px;"><strong>Ponavljanje zagrijavanja</strong> - Nakon otprilike minutu zagrijavanje se ponavlja još <strong>dva puta</strong> (ukupno <strong>trokratno zagrijavanje</strong>)</li>
      <li style="margin-bottom: 8px;"><strong>Ispiranje</strong> - Preparat se ispere <strong>kiselim alkoholom</strong> sve dok ne prestane izlaziti crvena boja</li>
      <li style="margin-bottom: 8px;"><strong>Prekid djelovanja alkohola</strong> - Ispiranje pod mlazom vode</li>
      <li style="margin-bottom: 8px;"><strong>Kontrastno bojenje</strong> - <strong>Metilensko modrilo</strong> nanosi se preko razmaza (obojit će stanice koje nisu acidorezistentne)</li>
      <li style="margin-bottom: 8px;"><strong>Završno ispiranje</strong> - Preparat se pere mlazom vode iz vodovoda</li>
      <li style="margin-bottom: 0;"><strong>Sušenje i mikroskopiranje</strong> - Preparat se posuši i mikroskopira</li>
    </ol>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #607d8b;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🔬</span>
      <h3 style="margin: 0; color: #455a64; font-size: 1.3rem; font-weight: 600;">Rezultati</h3>
    </div>
    <p style="margin: 0 0 15px 0; color: #2d3748; line-height: 1.6;">
      U preparatu će se vidjeti <strong>crveno obojene acidorezistentne bakterije</strong>, dok će ostale bakterije biti obojene plavo zbog metilenskog modrila.
    </p>
