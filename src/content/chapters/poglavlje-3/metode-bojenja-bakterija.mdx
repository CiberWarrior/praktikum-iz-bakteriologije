## Metode bojenja bakterija

      <h3 style="margin: 0; color: #1565c0; font-size: 1.3rem; font-weight: 600;">Priprema bakteriološkog preparata</h3>
    </div>
    <p style="margin: 0 0 15px 0; color: #2d3748; line-height: 1.6;">
      Oblik bakterijskih stanica može se odrediti mikroskopiranjem samo kad ih obojimo otopinama pojedinih bakterioloških boja. U tu svrhu potrebno je pripraviti tzv. bakteriološki preparat za mikroskopiranje.

    <div style="background: #f8f9fa; padding: 15px; border-radius: 8px;">
      <strong style="color: #1565c0; font-size: 14px; margin-bottom: 10px; display: block;">Tri su osnovna uvjeta za izradu dobrog bakteriološkog preparata:</strong>
      <ol style="margin: 0; padding-left: 20px; color: #2d3748;">
        <li style="margin-bottom: 8px;"><strong>Suha, odmašćena površina predmetnog stakalca</strong> (ispiranje predmetnice etanolom i provlačenje kroz plamen)</li>
        <li style="margin-bottom: 8px;"><strong>Što tanji razmaz bakterijskog materijala po stakalcu</strong></li>
        <li style="margin-bottom: 0;"><strong>Sušenje preparata na sobnoj temperaturi</strong></li>
      </ol>
    </div>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #ff9800;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🧫</span>
      <h3 style="margin: 0; color: #f57c00; font-size: 1.3rem; font-weight: 600;">Priprema iz bakterijske suspenzije</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Ako se preparat za bojanje bakterija pripravlja iz bakterijske suspenzije, uzorak se uzima bakteriološkom ušicom i tanko razvuče po predmetnom stakalcu.
    </p>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #4caf50;">
    <div style="display: flex; align-items: center; margin-bottom: 15px;">
      <span style="font-size: 20px; margin-right: 10px;">🦠</span>
      <h3 style="margin: 0; color: #388e3c; font-size: 1.3rem; font-weight: 600;">Priprema iz bakterijskih kolonija</h3>
    </div>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Ako se mikroskopiraju bakterijske kolonije s agarne podloge, na predmetnicu se stavi kap destilirane vode s dodatkom 0,3 % NaCl, bakteriološkom se ušicom uzme malo materijala kolonije, prenese se u kap i razvuče u što tanjem sloju na predmetnom stakalcu.
    </p>
  </div>

  <div style="background: rgba(255,255,255,0.9); padding: 20px; border-radius: 10px; margin-bottom: 20px; border-left: 4px solid #9c27b0;">

    </div>
    <p style="margin: 0 0 15px 0; color: #2d3748; line-height: 1.6;">
      Da se prilikom bojenja bakterije ne bi mehanički uklonile s predmetnog stakalca (oprale mlazom boje ili vode), osušeni razmaz fiksira se provlačenjem kroz plamen.
    </p>
    <p style="margin: 0; color: #2d3748; line-height: 1.6;">
      Fiksiranje preparata plamenom obavlja se tako da se predmetnica tri puta provuče kroz plamen Bunsenova plamenika, tako da je preparat s gornje strane, a svaki prolazak kroz plamen predmetnice traje oko 1 sekundu.

### Važne napomene

- Za sva bojenja bakterija čiste kulture bakterija moraju biti u **log fazi rasta**
- Pripremljeni mokri preparati osuše se na sobnoj temperaturi
- Fiksiranje je ključno za sprječavanje uklanjanja bakterija tijekom bojenja
